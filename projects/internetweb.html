Internet

Internet is a world wide interconnected system that uses established set of protocols to communicate between devices and users.

Web

Web is a hyperlink information system in which documents and resources are formatted using HTML where users can easily find, view and store data from the internet.

How are the Internet and Web different?

Internet is interconnected network of computer which is more hardware based, mean while the web is software service provided in the internet to browse media and resources.

ASCII and Unicode

ASCII, American Standard Code for Information Interchange, which uses 7 or 8 bit binary numbers, supports to encode English characters in the computer system. Unicode is universal character encoding system which supports to put characters of all languages in the computer system and is used by every internet platform.

What is UTF-8

UTF-8 is defined by Unicode system and, is a character encoding standard which is backward-compatible with ASCII and repensented with 1-4 bytes which makes it variable for all characters to be encoded in the computer system.

URL

URL,Uniform Resource Locator is a web address and crucical for the funtionality of internet and in enabling broswers to retrieve resources and servers and browsers to communicate because of its ability to point to a specific resource on the internet.

Domain Name

Domain name makes us possible to read a complex numeral IP address of a website because domain names are text we type into a machine to access a browser. That means domain names act as addresses for web servers.

HTTP

HTTP, the Hypertext Transfer Protocool, is a set of rules that dictate how information is exchanged on a network and is a foundation of the World Wide Web which enables communication between web servers an users.

HTTPS

HTTPS, the Hypertext Transfer Protocool Secure, is a combination of HTTP and Transport Layer Security which means it is a secure communication protocol that can encrypt data that is sent between websites and users.

IP address

IP or Internet Protocol address is the unique number of identifying that is assigned to every device connected to the internet.

DNS

Domain Name System makes it possible to type domain names to access websites instead of typing IP addresses because it is a system that translates domain names into IP addresses.

W3C Recommendations

A W3C Recommendation is a technical report or published document that defines web technologies and establishes best practices and technical specifications for web technologies.

XML, XHTML, HTML, HTML5

XML is markup language similiar to HTML but it does not predefine tags like HTML does to use. XHTML is an extensible and flexible HTML to work with other data formats. HTML, Hyper Text Markup Language, is a markup language that enables to write code in a text editor and it also defines the structure of the content of a web page. HTML5 is an advanced version of HTML which is also included JavaScript APIs to enhance multimedia, storage and hardware access.

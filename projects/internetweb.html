Internet

Internet is a world wide interconnected system that uses established set of protocols to communicate between devices and users.

Web

Web is a hyperlink information system in which documents and resources are formatted using HTML where users can easily find, view and store data from the internet.

How are the Internet and Web different?

Internet is interconnected network of computer which is more hardware based, mean while the web is software service provided in the internet to browse media and resources.

ASCII and Unicode

ASCII, American Standard Code for Information Interchange, which uses 7 or 8 bit binary numbers, supports to encode English characters in the computer system. Unicode is universal character encoding system system which supports to put characters of all languages in the computer system and used by every platform.

What is UTF-8

UTF-8 is defined by Unicode system and, is a character encoding standard which is backward-compatible with ASCII and repensented with 1-4 bytes which makes it variable for all characters to be encoded in the computer system.

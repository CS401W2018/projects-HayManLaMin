2.When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one that comes later takes priority. In this case, "flex-direction: row;" will take priority.

3. How do you change the CSS for a web page displayed on a phone?

We use @media ONLY screen AND to give specific width, and there, we make changes we want to do for our web page displayed on a phone.

Questions

1.What is normal flow in HTML?

Normal flow in HTML is the way of all elements are in arrange from top to buttom.

2.What does float do? Why do you need to clear the float?

Float can move a position of element on the page. For example, if we want our picture on the right and our passage side by it, we use float. To clear the float, we use div, slecter and clear property.

3.What do absolute, fixed, and relative do?

When we use absolute, the element is moved from the normal flow and we can shift it around. When we use relative, an element is stayed in the normal flow and we can also shift it around to left or right. Fixed shows an element according to the broswer window, the element stays in the same place no matter the size of the broswer.

4.How can you make nice navigation buttons for mobile devices?

By selecting navigation tag and using properties like background colors, padding, width and so on.

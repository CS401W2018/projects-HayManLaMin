Selectors

1.List out the different ways to select an item.

Tag Name, Descendent Rule, ID (no spaces), Class-multiple tags within same class. A tag can have more than Class adjective.

2.What is the difference between an ID and a Class?

In ID tag, we cannot put spaces but in class tag, we put more than one tags by putting spaces between them.

3.What is the difference between a SPAN tag and a DIV tag?

A span tag is used to edit an only specific element while a div tag is used to put some elements in a group.

4.What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A div tag has the ability to group elements for styling and layout while header tag, main or footer help SEO and they are more about the structure of the webpage.

5.What are the different ways to select a color?

Named colors, hexadecimal colors, RGB values, HSL colors and color picker.

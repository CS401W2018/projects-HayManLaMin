Questions

Explain the difference between an absolute link and a relative link. Why would you use relative links?
An absolute link is a link of other's web page that we copy and paste it in our site or page.
A relative link is a link of our own page we created that we copy and paste it in our new site we are creating.
List the 3 rules with relative links.
1. Same folder- file.html
2. Child folder(nephew)- folder/file.html
3. Up to parent folder - ../file.html
Do links always need to be inside a NAV tag?
Links do not always need to be inside a NAV tag. They can be in the main tags too.
